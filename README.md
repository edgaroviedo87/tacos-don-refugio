# Tacos Don Refugio — "El Placer del Sabor"

Marketing website for **Tacos Don Refugio**, a fictional traditional taquería in
**Villa de Álvarez, Colima, Mexico**. This is a portfolio demo — there is no real
business behind this brand.

## Status

Phase 1 — informational website (in planning). No online ordering yet; the architecture
is designed so ordering and a mobile app can be added later without a rewrite.

### Phase 1 scope
- **Menu** — tacos, tortas, órdenes (easily editable prices).
- **Avisos** — dated announcements (closures, price updates) that auto show/hide.
- **Gallery** — photos, focused on event catering.
- Branches, hours, maps, and call / WhatsApp CTAs.
- Strong local SEO and a WebGL hero with a robust non-WebGL fallback.
